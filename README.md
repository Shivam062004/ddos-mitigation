A Distributed Denial-of-Service (DDoS) attack is a malicious attempt to disrupt the normal functioning of a targeted server, service, or network by overwhelming it with a massive volume of traffic. These attacks typically originate from multiple compromised systems, making them difficult to detect and mitigate.

With the rapid growth of Software-Defined Networking (SDN), centralized control introduces new security challenges, especially against large-scale DDoS attacks. This project focuses on detecting and mitigating DDoS attacks in SDN environments using Machine Learning techniques.
